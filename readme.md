# RNN-LSTM-GRU
Recurrent Neural Network, Long Short-Term Memory and Gated Recurrent Unit Algorithms. These natural language processing algorithms are used to process sequential data.
## Project Structure

```
.
├── data
│   └── MNIST 
│   └── test.csv  
│   └── train.csv
├── custom_models.py
├── main.py
├── README.md
├── requirements.txt
├── text_classification_tweets.py
```
## Usage

```
python3 train.py 
python3 main.py 
```
